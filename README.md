我在 Jide 的时间里，我的老板多次提到shell编程的重要性。学习shell，当然得看官方文档，我惊奇的发现，github中没有关于这方面的翻译资料，想想国内对shell并不是很重视（学校里面没开设这方面的课程）而这个确实有用，所以还是边学边翻译吧，帮助自己又可能帮助别人，嘿，一举俩得。
